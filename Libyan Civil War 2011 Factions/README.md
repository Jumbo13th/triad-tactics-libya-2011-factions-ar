## Overview

This addon provides factions for the 2011 Libyan Civil War in Arma Reforger.
It was created specifically for the Sirte, Libya terrain: [Sirte by Triad Tactics Server](https://reforger.armaplatform.com/workshop/66E15BA754C92696-SirtebyTriadTacticsServer).
Terrain source and updates: [github.com/Jumbo13th/triad-tactics-sirte-ar](https://github.com/Jumbo13th/triad-tactics-sirte-ar)

> [!WARNING]
> **Land-Rover with ZU-23-2:** This vehicle does not work correctly in multiplayer scenarios. It is not recommended to use it.
